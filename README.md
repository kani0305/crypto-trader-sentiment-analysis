# 📊 Crypto Trader Behavior Analysis

## 📌 Overview
This project analyzes the relationship between trader performance and Bitcoin market sentiment (Fear vs Greed Index).

## 🎯 Objectives
- Understand how sentiment affects trading outcomes
- Identify profitable trading patterns
- Build an interactive dashboard

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Streamlit (Dashboard)

## 📈 Key Insights
- Traders behave differently in Fear vs Greed markets
- Risk-taking increases during Greed phases
- Profitability varies based on sentiment

## 🚀 Dashboard
Run locally:
streamlit run app.py

## 📁 Project Structure
- main.py → Data cleaning & merging
- save_charts.py → Chart generation
- app.py → Interactive dashboard
- charts/ → Visual insights
